# 🐄 Qurbani-hatt

A modern web platform to browse, buy, and sell Qurbani animals بسهولة and with confidence. Built with a focus on clean UI, trust, and smooth user experience.

---

## 🚀 Features

* 🐐 Browse Qurbani animals (Cow, Goat, Sheep)
* 🏷️ Detailed animal listings with images and descriptions
* 🧾 Qurbani Tips section for guidance
* 🌟 Top Breeds showcase
* 🔄 Smooth scroll navigation (single-page UX)
* 📱 Fully responsive design

---

⚙️ Installation
# Clone the repo
git clone (https://github.com/Tah56/QurbanirHatt.git](https://github.com/Tah56/QurbanirHatt.git)

# Go to project folder
cd qurbani-marketplace

# Install dependencies
npm install
---
# Run development server
npm run dev
🌐 Usage
Open http://localhost:3000
Browse animals
Click buttons to scroll to sections
Explore tips and breeds

* **UI Components:** hero/ui
